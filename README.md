# py-qa-engine

Unified, modular test automation framework built with Python, Playwright, Appium, and Behave (BDD). Developed inside Docker containers on Ubuntu 20.04, featuring Allure Reporting and CI/CD integration.

## 🚀 Project Overview
This repository documents my transition from TypeScript to an advanced Python QA Automation Engineer. The core architecture focuses on reusable components, the Page Object Pattern, and data-driven testing, completely isolated within Docker environments.

## 🗺️ Roadmap & Tech Stack
- **Languages:** Python (with Type Hints), SQL
- **UI Automation:** Playwright (Python) & Appium (Android)
- **API & Contract Testing:** Requests, HTTPX, Pact Python
- **BDD Layer:** Behave (Gherkin)
- **Reporting:** Allure Report
- **CI/CD:** GitLab CI, GitHub Actions, Jenkins
- **OS & Env:** Ubuntu 20.04 LTS, Docker, docker-compose, venv
